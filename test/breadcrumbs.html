Styling with Mixins

We've included Scss mixins and HTML classes that can be used to style breadcrumbs. If you are using the mixins, you may include the styles on whatever class or ID you'd like. If you are using vanilla CSS, you'll only have the default classes available (which we cover below).

Base Mixin

Use this mixin to create the base styles for breadcrumbs. This will include styles that create the basic structure of a breadcrumb. You'll want to make sure you include a container, we recommend an <ul>. Inside that container include list items that will contain your crumbs.

.breadcrumbs { @include crumb-container; }
This is the base mixin.
Background Mixin

You can control the background, border and text styles for the alert using this mixin. We base the border and text style on the background color you choose using some sweet Sass logic.

.alert-box {
  @include alert-base;
  @include alert-bg($bg);
}
This is base and bg mixins.

You can set $bg color to any of our predefined color variables or any hexadecimal color of your choosing. To set this value to something other than our default, use @include alert-bg(#hexcolor);

The Close Button Mixin

You can include a close button with each of the alert boxes by adding an anchor right before the closing </div>. This anchor should have a class of .close in order to work with foundation.alerts.js, which contains the interaction for closing.

.alert-box {
  @include alert-base;
  @include alert-bg;
  .close { @include alert-close; }
}
This is base, bg and close mixins. ×
Radius Mixin

We've also created a global radius mixin that you can use to add border-radius to any element you'd like.

.alert-box {
  @include alert-base;
  @include alert-bg($success-color);
  @include radius($radius);
}
This is base, bg and radius mixins. ×
Default Scss Variables
// We use this to control alert padding.
$alert-padding-top:         emCalc(11px)
$alert-padding-left:        $alert-padding-top
$alert-padding-right:       $alert-padding-top + emCalc(10px)
$alert-padding-bottom:      $alert-padding-top + emCalc(1px)

// We use these to control text style.
$alert-font-weight:         bold
$alert-font-size:           emCalc(14px)
$alert-font-color:          #fff
$alert-font-color-alt:      darken($secondary-color, 60%)

// We use this for close hover effect.
$alert-function-factor:     10%

// We use these to control border styles.
$alert-border-style:        solid
$alert-border-width:        1px
$alert-border-color:        darken($primary-color, $alert-function-factor)
$alert-bottom-margin:       emCalc(20px)

// We use these to style the close buttons
$alert-close-color:         #333
$alert-close-position:      emCalc(5px)
$alert-close-font-size:     emCalc(22px)
$alert-close-opacity:       0.3
$alert-close-opacity-hover: 0.5
$alert-close-padding:       5px 4px 4px

// We use this to control border radius
$alert-radius:              $global-radius

Note: emCalc(); is a function we wrote to convert px to em.

Styling with CSS

If you don't want to use Scss, you're still covered. We include presentational classes that will quickly style your alert boxes, it was used in the first snippet on this page. The default class for the container is .alert-box. This can be applied to any block level element, like <div>, <section> or <article>. Inside of this container you can include <a href="" class="close">&times;</a> to create a close button that sticks to the top right of the alert-box.
