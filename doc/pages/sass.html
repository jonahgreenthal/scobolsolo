--- title: Getting Started With Sass ---

Sass is the easiest way to customize Foundation! It is a superset of CSS that adds in amazing features such as variables, nested selectors and loops.

***

SCSS: How it works

SCSS is a great way to speed up front-end development. SCSS (Sassy CSS) syntax was introduced as "the new main syntax" for Sass and builds on the existing syntax of CSS. We created a SCSS version to allow more friendly and dynamic styling, but they also offer greater customization in the SCSS version than in the HTML/CSS version

Comand Line Interface: More Power for You!

We created this command line interface (CLI) to make creating a Foundation project and updating your project. Using this CLI will greatly speed up your workflow as it does many tasks for you. When updating the CLI updates your project while leaving your customized styles in tact. This is the recommended way to start.

How to install the CLI

Using the CLI to install Foundation 5 only takes two easy commands. You only have to install the CLI once.

Bash

{{#markdown}} ``` npm install -g bower grunt-cli ``` {{/markdown}}

We use Bower to manage updates that we commit to the Foundation GIT.

Bash

{{#markdown}} ``` gem install foundation ``` {{/markdown}}

Install the Foundation on your computer only takes one command function.

If you run into permission errors running these commands in the CLI you can add the word 'sudo' before the line inside the interface:

Bash

{{#markdown}} ``` sudo npm install -g bower grunt-cli ``` {{/markdown}}

Bash

{{#markdown}} ``` sudo gem install foundation ``` {{/markdown}}

Compiling Sass into CSS & Creating a Project

There are two easy ways our CLI helps you compile Sass: Compass or Libsass with Grunt.

Compass

Creating a Compass project

Step 1: Compass is the easiest way to get going with Sass. If you do not already have installed here it can be installed with the following command:

{{#markdown}} ``` gem install compass ``` {{/markdown}}

Step 2: Next we'll use the Foundation CLI to create a new project:

{{#markdown}} ``` foundation new MY_PROJECT ``` {{/markdown}}

Step 3: Boom, your project is created! Now updating Found will be easy.

{{#markdown}} ``` cd MY_PROJECT foundation update ``` {{/markdown}}

Now You're Ready to Make Changes

Make sure to run ```compass watch``` to compile your SCSS in CSS.

Libsass with Grunt

Creating a Libsass project

Step 1: First, navigate into the directory where you want to create your project.

{{#markdown}} ``` cd path/to/sites ``` {{/markdown}}

Step 2: Next we'll use the Foundation CLI to create a new project:

{{#markdown}} ``` foundation new project_name --libsass ``` {{/markdown}}

Step 3: Boom, your project is created! Now you need to use Grunt to compile everything before you start working:

{{#markdown}} ``` cd project_name grunt build ``` {{/markdown}}

Now You're Ready to Make Changes

Make sure to run the command ```grunt``` to compile your SCSS in CSS.
