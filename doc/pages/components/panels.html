This allows you to view your page sections as you add content to them or add emphasis to a section. The mixin contains all the necessary styles and they can be customized a bit within the mixin. The rest of the important styles are controlled by variables. The code looks like this: {{#markdown}} ```scss .your-class-name { @include panel; } ``` {{/markdown}}
Panel content goes here...
You also have **three options** available to customize the panel within the mixin. These control the background color (which also effect border and font color) and the interior padding of the panel itself. {{#markdown}} ```scss .your-class-name { @include panel($bg, $padding, $adjust); } /* This controls the background color, border color and type color based on brightness of the bg. */ $bg: $secondary-color /* This controls the interior padding */ $padding: $panel-padding /* Control whether panel mixin adjusts header styles */ $adjust: true ``` {{/markdown}}
