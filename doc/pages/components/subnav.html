To use the mixin, you'll need to have the [extension installed]({{relative absolute 'dist/docs/components/sass.html'}}) or grab [_variables.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/_variables.scss), [_global.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/components/_global.scss) and [_sub-nav.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/components/_sub-nav.scss) from Github and throw them into a Foundation folder in your project directory. From there, you can import the files at the top of your own SCSS stylesheet, like so: {{#markdown}} ```scss @import "foundation/components/sub-nav"; ``` {{/markdown}} ## Configuring the Sub Nav You can build your sub-nav using our global mixin by including it on a custom class or ID in your own stylesheet. The mixin contains options for changing the font color, font size and active background color. The rest of the defaults can be modified using the available variables. The global mixin looks like this:

SCSS

{{#markdown}} ```scss /* Using the default styles */ .basic-subnav-class { @include sub-nav(); } ``` {{/markdown}}

There are also three options you can customize on the fly when writing this mixin. These controls things like: divider color, text size and font color.

SCSS

{{#markdown}} ```scss /* Using the custom styles */ .custom-subnav-class { @include sub-nav( $font-color:$primary-color, $font-size:1.2em, $active-bg:$primary-color); } ``` {{/markdown}}
