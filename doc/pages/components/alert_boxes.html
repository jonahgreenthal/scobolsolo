--- layout: doc/layouts/default.html --- # Alert Boxes #### Alerts are a handy element you can drop into a form or inline on a page to communicate success, warnings, failure or just information. They'll conform to 100% of the container width you put them in. {{> examples_alert_basic}} *** ### How To Work With Alerts There are two ways to build alerts. You can use our predefined HTML classes or use our crafted mixins. Building an alert using our predefined classes is super-easy, you only need a block-level element with a class of `.alert-box` (we usually use a `
`). You may also include an anchor with a class of `.close` to create a close box. Here are some examples: ### HTML {{> examples_alert_basic_rendered}} For these styles to come across, make sure you have the **default Foundation CSS package** or that you've selected **alerts from a custom package**. These should be linked up following our default [HTML page structure]({{relative absolute 'dist/docs/components/getting_started.html'}}). *** ### Build with Mixins We've included SCSS mixins used to style alert boxes. To use these mixins, you'll need to have the [extension installed]({{relative absolute 'dist/docs/components/sass.html'}}) or grab [_variables.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/_variables.scss), [_global.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/components/_global.scss) and [_alert-boxes.scss](https://github.com/zurb/foundation/blob/master/scss/foundation/components/_alert-boxes.scss) from Github and throw them into a Foundation folder in your project directory. From there, you can import the files at the top of your own SCSS stylesheet, like so: {{#markdown}} ```scss @import "foundation/variables"; @import "foundation/components/global"; @import "foundation/components/alert-boxes"; ``` {{/markdown}} If you are using the mixins, you may include the styles on whatever class or ID you'd like, just make sure you follow our markup structure: {{> examples_alert_mixin_rendered}} ## Configuring Alert Boxes You can use this to quickly create alerts with a single mixin.

SCSS

{{#markdown}} ```scss /* Using the default styles */ .your-class-name { @include alert(); } /* Using the available options */ .your-class-name { @include alert($bg:#ff6c3c); } ``` {{/markdown}}

Before you can use the top bar you'll want to verify that both jQuery (or Zepto) and `foundation.js` are available on your page. You can refer to the [javascript documentation](../javascript.html) on setting that up.
Just add `foundation.alerts.js` AFTER the `foundation.js` file.
