The global mixin will create the necessary style for the button. The global mixin looks like this: There are also **six options** you can customize on the fly when writing this mixin. These controls things like: background color, border color, text color, size and state. Setting any of these options to **false** will negate those styles. {{> examples_buttons_custom_mixin}} ##### Base Mixin You also have access to a few internal mixins that can create parts of the button as needed. The base mixin will create a button base that only include structural styles. {{> examples_buttons_base_mixin}} ##### Size Mixin The size mixin will let you control the padding, which adjusts text size. You can also choose whether the button is full-width or an input button. {{> examples_buttons_size_mixin}} ##### Style Mixin The last internal mixin you have access to for buttons is the style mixin. This will help you style background color, text, border and radius. {{> examples_buttons_style_mixin}} Inset Shadow, Border Radius & Transition Mixin Sometimes you want to add a nice fancy shine to the edge of your button. And sometimes you want to make that shine look like it gets depressed upon tap or click. We've got you covered with our quick inset shadow mixin. You can use our radius mixin to quickly apply rounded corners or a transition mixin to give the button a nice zero-fade. {{> examples_buttons_options_mixin}} **Note:** `em-calc();` is a function we wrote to convert `px` to `em`.
