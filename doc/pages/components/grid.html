--- layout: doc/layouts/default.html --- # Grid

Create powerful multi-device layouts quickly and easily with the 12-column, nestable Foundation grid. If you're familiar with grid systems, you'll feel right at home. If not, you'll learn quickly.

*** ### Basic * **Foundation uses border-box**: Borders and padding do not affect the overall width of the columns, making the math dead-simple. * **Foundation is mobile-first**. Code for small first, and larger devices will inherit those styles. Customize for larger screens as necessary.
