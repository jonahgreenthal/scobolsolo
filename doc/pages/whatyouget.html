--- title: What You Get ---

We've put together all of the pieces you need in one package, let's see what's inside.

Customization

  • app.css
    Here you can determine what stylings will be in your project. By default all of Foundation is imported, but you can deactivate and select specific styles you want to import.
  • _settings.scss
    Here you can activate preselected customization variables and customize Sass where it will be compiled into CSS.
  • app.cssWe include a stylesheets folder where you can customize your CSS. This files has basic Foundation CSS styling and can be customized here. Add any additional css style sheets in this folder.

JavaScript

  • app.js
    This is where you should put your javascript plugins or custom scripts to be imported into your index file. We do this to keep the markup on the html clean and simple by keeping your JS in a separate place.
  • index.html
    Your Index file should import both the minified versions of the Foundation JavaScript and the jQuery as well as the app.js.

  • Any changes to the customizations in this folder will be overwritten when your file is compiled into CSS.
